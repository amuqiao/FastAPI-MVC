# 知识库

AI 架构师知识体系，覆盖深度学习理论、AI 工程实践、软件工程、基础设施等方向。

## 入库规则

新文档进来，对照下表决定放哪里。**不确定就先放 `0.收件箱/`**。

| 文档内容 | 放入目录 |
|---|---|
| 某个模型或算法的原理解析 | `1.深度学习/` 对应子目录 |
| 模型训练、部署、框架使用的工程实践 | `2.AI工程/` 对应子目录 |
| 与 AI 无关的后端、前端、架构知识 | `3.软件工程/` |
| Docker、K8s、Git、环境配置 | `4.基础设施/` |
| 面试题、岗位分析、模拟问答 | `5.面试/` |
| 用来生成或整理笔记的 Prompt 模板 | `6.提示词工具箱/` |
| 某个项目的架构设计或技术方案 | `7.项目文档/` |
| 外链、书单、GitHub 项目清单 | `8.资源/` |
| **不确定放哪** | `0.收件箱/`，定期处理 |

---

## 分层规则

- **最多 3 层**：根目录 → 主题子目录 → 文件
- `2.AI工程/` 下按职责分目录（训练/优化/部署/Agent/RAG/平台），不以框架名建目录
- `1.深度学习/` 与 `2.AI工程/` 的边界：原理归深度学习，落地实现归 AI 工程
