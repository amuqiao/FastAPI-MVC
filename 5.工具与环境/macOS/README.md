# macOS

> **主题定位**：本目录用于沉淀与 macOS 日常使用相关的知识，重点覆盖常用操作、问题排查、软件获取与安装、推荐应用。  
> **主题类型**：操作型 + 推荐型。  
> **当前状态**：已建立基础骨架，后续按主题增量维护。

## 1. 这个主题解决什么问题

这个主题不追求讲清 macOS 的底层原理，而是解决：

- 常见操作怎么做
- 常见问题怎么排查
- 软件通常从哪里获取
- 哪些应用值得长期保留

因此它不适合强行拆成“基础概念 / 核心原理 / 工程实践”。  
更合理的是围绕“操作、应用、排障”组织。

## 2. 推荐阅读顺序

如果你是第一次进入这个主题，按下面顺序看：

1. `macOS-主文档.md`
2. `macOS-常用操作.md`
3. `macOS-推荐应用.md`
4. 再按具体问题进入单项指南

## 3. 按问题找文档

如果你想快速定位：

- 想看这个主题的总览和维护规则：`macOS-主文档.md`
- 想看日常操作入口：`macOS-常用操作.md`
- 想看应用、软件获取与推荐：`macOS-推荐应用.md`
- 想切换 Shell：`macOS Shell 切换指南.md`
- 想重置网络：`macOS 网络重置指南.md`

## 4. 维护规则

- 新增日常操作，优先补到 `macOS-常用操作.md` 或对应专项指南
- 新增应用推荐，优先补到 `macOS-推荐应用.md`
- 只有整体理解变化时，才改 `macOS-主文档.md`
- 只有导航变化时，才改 `README.md`
- 不确定、版本敏感、待验证的内容，单独标注，不直接写成正式结论
