# macOS-主文档

## 1. 文档目标

本文是 `macOS` 主题的总纲，只负责：

- 说明主题边界
- 提供阅读路径
- 收敛稳定结论
- 索引到子文档

不负责承载所有具体操作细节。

## 2. 主题边界

当前 `macOS` 主题重点覆盖：

- 日常高频操作
- 常见问题排查
- 软件获取与安装原则
- 常用应用推荐

当前不重点覆盖：

- macOS 内核或系统原理
- 版本演进史
- 深入底层机制分析

## 3. 主题类型判断

这个主题属于：

- 操作型
- 推荐型

必要时可带少量排障内容，但不应被整理成“原理型主题”。

## 4. 当前模块设计

当前按下面几个模块维护：

- `macOS-常用操作.md`
  负责日常操作入口和问题定位
- `macOS-推荐应用.md`
  负责软件获取原则、应用分类、推荐清单
- `macOS Shell 切换指南.md`
  负责 Shell 切换专项操作
- `macOS 网络重置指南.md`
  负责网络重置专项操作

## 5. 稳定维护结论

1. 这个主题应围绕“操作、应用、排障”组织，而不是围绕“概念、原理、工程实践”组织。
2. 具体步骤优先沉淀到子文档，不把主文档写成操作手册。
3. 软件推荐要优先保留“适用场景”和“选择理由”，而不是只堆链接。
4. 高风险、版本敏感、待验证内容应单独标注，不直接视为稳定结论。

## 6. 后续扩展方向

后续如果内容继续增长，可按需新增：

- `macOS-排障清单.md`
- `macOS-快捷键与效率设置.md`
- `macOS-开发环境准备.md`

不符合长期增长趋势时，不要为了完整而预先建空文件。
