# 分析深度档位

本 skill 默认支持三档深度：

- `scan`
  - 用于快速建立整体认知
  - 默认输出项目定位、技术栈、主模块、主链路、2 到 3 个关键设计决策
  - 默认总图数为 1 到 2 张

- `architecture`
  - 用于深入理解整体架构
  - 默认输出模块依赖、核心链路、设计决策、数据模型、扩展点、风险点
  - 默认总图数为 2 到 4 张

- `module`
  - 用于聚焦单个模块
  - 默认输出模块职责边界、入口、内部链路、依赖、数据结构、边界情况、最小改动路径
  - 默认总图数为 1 到 3 张

如果用户没有明确指定深度，默认使用 `scan`。
