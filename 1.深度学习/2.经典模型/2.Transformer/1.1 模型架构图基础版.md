# Transformer 极简核心总结

1. **定位**：**序列到序列** SOTA 模型，解决长距离依赖建模难题
2. **核心Backbone**：**编码器-解码器**结构，每层包含自注意力和前馈网络
3. **最大创新**
    - **自注意力机制**：直接计算序列中任意两个位置的依赖关系
    - **位置编码**：显式注入序列位置信息
    - **多头注意力**：并行学习多维度特征表示
    - **并行计算**：摆脱RNN的顺序计算限制
4. **结构范式**
输入 → 嵌入+位置编码 → 编码器（自注意力+前馈）→ 解码器（掩码自注意力+交叉注意力+前馈）→ 线性层+Softmax → 输出
