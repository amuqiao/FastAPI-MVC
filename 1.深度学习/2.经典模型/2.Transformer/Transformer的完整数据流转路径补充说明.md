# Transformer 完整数据流转路径（含维度）- 示例说明
我们以**机器翻译场景**为例，设定Transformer的核心参数（贴合论文经典配置）：
- 批量大小 `batch = 32`（一次处理32个样本）
- 源序列长度 `src_len = 20`（如英文句子最长20个词）
- 目标序列长度 `tgt_len = 15`（如中文翻译结果最长15个词）
- 模型维度 `d_model = 512`（Transformer论文核心维度）
- 词表大小 `vocab_size = 10000`（涵盖常用中英文词汇）
- 编码器/解码器层数 `N = 6`（论文标准层数）

## 1. 编码器路径（数值化维度示例）
```
源序列 [32, 20] （32个英文句子，每个句子20个词索引）
    ↓
词元嵌入 [32, 20, 512] + 位置编码 [32, 20, 512]
    ↓
特征融合 [32, 20, 512] （逐元素相加，维度不变）
    ↓
编码器层（6层堆叠，每层输出均为 [32, 20, 512]）
    ├─ 多头自注意力：输入[32,20,512] → 输出[32,20,512]
    ├─ Add & Norm：输入[32,20,512] → 输出[32,20,512]
    ├─ 前馈网络：输入[32,20,512] → 升维[32,20,2048] → 降维[32,20,512]
    └─ Add & Norm：输入[32,20,512] → 输出[32,20,512]
    ↓
编码器最终输出 [32, 20, 512]
```

## 2. 解码器路径（数值化维度示例）
```
目标序列 [32, 15] （32个中文翻译前缀，每个前缀15个词索引）
    ↓
词元嵌入 [32, 15, 512] + 位置编码 [32, 15, 512]
    ↓
特征融合 [32, 15, 512]
    ↓
解码器层（6层堆叠）
    ├─ 掩码多头自注意力：输入[32,15,512] → 输出[32,15,512]（掩码屏蔽未来词）
    ├─ Add & Norm：输入[32,15,512] → 输出[32,15,512]
    ├─ 多头交叉注意力：Query[32,15,512] + Key/Value[32,20,512] → 输出[32,15,512]
    ├─ Add & Norm：输入[32,15,512] → 输出[32,15,512]
    ├─ 前馈网络：输入[32,15,512] → 升维[32,15,2048] → 降维[32,15,512]
    └─ Add & Norm：输入[32,15,512] → 输出[32,15,512]
    ↓
解码器最终输出 [32, 15, 512]
```

## 3. 输出路径（数值化维度示例）
```
解码器输出 [32, 15, 512]
    ↓
线性层 [32, 15, 512] → [32, 15, 10000]（映射到词表空间）
    ↓
Softmax [32, 15, 10000]（每个位置输出10000个词的概率）
    ↓
预测结果 [32, 15]（取概率最大的词索引，完成翻译）
```

## 快速预览（一行式数值化维度）
源序列 [32, 20] → 嵌入 [32, 20, 512] → 编码器 [32, 20, 512] → 交叉注意力 ← 目标序列 [32, 15] → 嵌入 [32, 15, 512] → 掩码自注意力 → 解码器 [32, 15, 512] → 线性层 [32, 15, 10000] → 预测 [32, 15]

## 关键维度说明
- `d_model=512` 是贯穿编码器/解码器的核心维度，除前馈网络升维（4×d_model=2048）外，其余核心层均保持该维度，保证残差连接的可行性；
- 交叉注意力中，Key/Value来自编码器（[32,20,512]）、Query来自解码器（[32,15,512]），但输出维度始终匹配Query的序列长度（15），实现“目标序列关注源序列”；
- 最终线性层将 `d_model=512` 映射到 `vocab_size=10000`，是模型从“语义表示”到“具体词预测”的关键转换。

### 总结
1. Transformer核心维度参数（示例）：`batch=32`、`d_model=512`、`vocab_size=10000`，前馈网络固定升维为`4×d_model`；
2. 编码器/解码器的核心层（自注意力、交叉注意力、Add&Norm）均不改变序列长度和模型维度，仅前馈网络临时升维；
3. 交叉注意力能兼容源/目标序列的不同长度（如20和15），最终输出维度匹配目标序列长度，是跨序列注意力的核心特性。
