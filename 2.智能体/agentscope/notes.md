
# 什么是 Tune（微调）

## 微调的目的
为了提升 Agent 应用完成任务的能力

## 微调的定义
对 Agent 应用中使用的模型进行小规模训练的技术

## 常见的微调方法

### 1. 监督微调（SFT, Supervised Fine-tuning）
- 通过高质量数据
- 指导模型应该做什么

### 2. 强化微调（RFT, Reinforcement Fine-tuning）
- 让模型自行探索
- 在探索过程中或结束后给予奖励
