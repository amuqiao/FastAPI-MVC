## 一、基础信息

- **任务**：机器翻译、文本摘要、问答系统、时序预测（通用序列到序列模型）
- **核心定位**：**抛弃RNN，用自注意力做序列SOTA**
- **对比**：并行计算速度远快于RNN/LSTM，能更好捕捉长距离依赖

```mermaid
    D[编码器-解码器结构]:::coreStyle
    
    A1[直接计算任意位置依赖]:::descStyle
    B1[显式注入位置信息]:::descStyle
    C1[并行学习多维度特征]:::descStyle
    D1[分离输入处理和输出生成]:::descStyle
    
    A --> A1
    B --> B1
    C --> C1
    D --> D1
```

1. **自注意力机制**（最核心）
直接计算序列中任意两个位置的依赖关系，不受距离限制。
2. **位置编码**
显式注入序列位置信息，解决自注意力无法感知顺序的问题。
3. **多头注意力**
并行学习多维度特征表示，捕获不同角度的依赖关系。
4. **编码器-解码器结构**
分离输入处理和输出生成，更适合序列到序列任务。

4. 残差连接 + 层归一化
5. 前馈网络
6. 残差连接 + 层归一化
