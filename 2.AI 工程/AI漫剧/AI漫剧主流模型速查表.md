# AI 漫剧主流模型速查表

> **文档职责**：收录 AI 漫剧制作各阶段主流模型，按厂商分类列表，含一句话简介和关键参数
> **适用场景**：选型时横向对比同阶段模型；迭代时快速判断新模型是否有替换价值
> **目标读者**：平台开发者，已了解基本架构，需要快速定位和对比具体模型
> **维护规范**：新模型上线后在对应阶段表格末尾追加行并注明日期；价格变动直接修改价格列；模型下线或停止维护时在名称后标注 `[停更 YYYY-MM]`；能力有重大升级时更新"核心优势"列并注明版本

## 三、分镜一致性辅助：LoRA 训练平台

> 适用路线 B（LoRA 微调）。训练一次，该角色后续所有分镜图生成均使用该权重。

| 平台 / 工具 | 类型 | 核心优势 | 训练成本 | 输出格式 | 本地替换 |
|------------|------|---------|---------|---------|---------|
| **Replicate** | 云端 API | 最易集成，API 一键提交训练任务，无需 GPU | ~$0.5–2 / 次 | `.safetensors` | ✅ Kohya |
| **fal.ai 训练** | 云端 API | 与 fal.ai 图像生成直接联动，训练完即可调用 | ~$1–3 / 次 | FLUX LoRA | ✅ SimpleTuner |
| **Kohya SS** 🔓 | 本地开源 | 最灵活，参数完全可控，是云端方案的本地替代 | GPU 成本 | `.safetensors` | — 本身即开源 |
| **SimpleTuner** 🔓 | 本地开源 | 专为 FLUX LoRA 优化，训练速度比 Kohya 快 | GPU 成本 | FLUX LoRA | — 本身即开源 |

> 训练素材要求：同一角色 20–50 张图，多姿态 / 多角度 / 多表情，避免重复构图。
