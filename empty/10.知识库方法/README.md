# 知识库方法

> 定位：本目录用于存放与具体主题解耦的知识库方法层文档，包括构建规范、维护清单和提示词模板。

---

## 1. 这个目录解决什么问题

这里不存放具体主题知识，而存放“如何构建和维护知识库”的通用规则。

因此，这个目录适合作为：

- 所有主题知识库的上位方法层
- 新主题搭建时的参考入口
- 后续维护和更新时的统一规则来源

## 3. 推荐使用顺序

### 场景一：第一次搭建知识库

按这个顺序用：

1. 先看 `知识库构建与维护规范`
2. 再看 `知识库极简操作清单`
3. 最后从 `知识库提示词模板库` 里选模板

### 场景二：已有知识库，想新增内容

按这个顺序用：

1. 先看 `知识库极简操作清单`
2. 再从 `知识库提示词模板库` 中选更新模板

### 场景三：想检查知识库是否开始失控

按这个顺序用：

1. 先看 `知识库极简操作清单`
2. 再用 `知识库提示词模板库` 里的体检模板
3. 如有必要，再回看 `知识库构建与维护规范`

## 4. 维护原则

维护这个目录时，建议遵循四条原则：

1. 这里只放通用方法，不放具体主题内容。
2. 主题知识库中不重复堆放这些通用规则，优先引用这里。
3. 新增方法文档时，优先补充到这个目录，而不是散放在根目录。
4. 如果方法本身发生变化，先更新这里，再决定是否同步影响主题知识库。
