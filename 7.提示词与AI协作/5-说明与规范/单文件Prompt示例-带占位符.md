# 单文件Prompt示例-带占位符

## 用途

这是单文件复用型 Prompt 的示例。适合高频复用、带少量占位符、单文件就能工作的场景。

## Prompt

```text
请围绕“{主题或对象}”完成这次任务。

任务目标：
- {目标1}
- {目标2}

输出要求：
- 先回答最核心的问题：{核心问题}
- 默认不要扩写成长篇报告
- 如果需要举例，优先给 {示例数量} 个
- 如果需要作图，限定为 {图数量} 张

约束：
- 不要偏离“{范围边界}”
- 不要展开与当前任务无关的背景
- 如果信息不足，先列出最小必要假设
```

## 说明

当一个 Prompt 满足下面这些特征时，优先考虑放到“单文件模板”而不是“专题模块”：
- 复用频率高
- 单文件足够表达清楚
- 只需要少量占位符
- 不需要单独维护 references
